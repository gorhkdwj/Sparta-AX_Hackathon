# ㅡ EVALUATION

## AX 해커톤 평가 기준

**척도 1점(매우 미흡) ~ 5점(매우 우수) · 두 관점에서 평가합니다.**

### 커리어개발 <span style="color:#d9534f">40점</span>

* **A1 · 문제 정의**
  * 문제를 자기 말로 재정의하고, 접근한 근거를 설명하는가
* **A2 · 구조화**
  * 결과물·발표자료의 전달력(가독성·구조화) — 전원 공통
* **A3 · 주도성**
  * 요구를 넘어 실무 디테일을 주도적으로 고민·제안했는가

### AI테크 <span style="color:#d9534f">60점</span>

* **B1 · 문제 정의**
  * 문제를 자기 말로 재정의하고, 접근한 근거를 설명하는가
* **B2 · 기술적 문제해결**
  * 막힌 지점·기술 난관을 타당하게 풀었는가 (접근의 합리성)
* **B3 · 결과물 완성도·동작**
  * basic -> standard -> challenge 중 어디까지 동작하는가
