# Phase 1 · 데이터 분석과 예외 기준 설계

## 목표

`키즐링/data/leads_sample.csv`를 분석해 이메일 생성과 캠페인 기획에 필요한 데이터 구조, 분포, 예외 케이스를 확정한다.

## 입력 자료

- `키즐링/data/leads_sample.csv`
- `.claude/skills/kizzling-analyze-leads/SKILL.md`
- 제공 참고 자료: `키즐링/.claude/skills/analyze.md`
- `docs/requirements-contract.md`

## 수행 계획

1. CSV 필수 컬럼을 확인한다.
2. 전체 행 수와 직책 분포를 확인한다.
   - 원장
   - 팀장
   - 사업개발
3. `org_type`, `interest_feature`, `follow_up_priority` 분포를 집계한다.
4. 이메일 결측 리드를 찾는다.
5. 상담 메모 결측·모호 리드를 찾는다.
6. 완전 중복 리드를 찾는다.
7. Basic 생성 대상, review 대상, 발송 불가 후속 메모 대상, 중복 제외 대상을 분리한다.
8. 캠페인 세그먼트 후보를 도출할 수 있도록 분포 요약을 남긴다.

## 산출물

- 데이터 분석 요약
- 예외 리드 목록
- 중복 리드 처리 기준
- 캠페인 세그먼트 후보 메모
