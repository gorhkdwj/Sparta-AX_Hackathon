# Public Link Prep

Basic 단계 제출용 노션 페이지 또는 구글 시트 공개 링크 생성을 위한 준비 메모입니다.

## 업로드 대상

- `output/email_drafts.csv` (49행, 컬럼: `lead_id`, `subject`, `body`, `review_flag`)

## 권장 절차

1. Google Drive(또는 노션)에서 새 스프레드시트(또는 페이지)를 만든다.
2. `email_drafts.csv`를 가져오기 한다.
3. 헤더가 `lead_id`, `subject`, `body`, `review_flag` 순서인지 확인한다.
4. 공유 설정을 "링크가 있는 사용자는 볼 수 있음"으로 변경한다.
5. 공유 URL을 `output/submission-summary.md`의 "공개 링크" 항목에 기록하고, 링크 접근 가능 여부를 직접 열어 확인한다.

## 미완료 항목

- 실제 공개 URL 생성은 외부 Google/노션 계정 작업이 필요해 이 세션에서는 완료하지 못했다. 제출 직전 위 절차대로 담당자가 직접 생성·확인해야 한다.
